Firewall

Pokud ti to nefunguje, bude to asi firewallem.🤔

  1. otevři Ovládací panely → Systém a zabezpečení → Firewall v programu Windows Defender (povolit aplikaci nebo funkci průchod přes Firewall v programu Windows Defender)
  2. musíš povolit jinou aplikaci
  3. klikni na tlačítko procházet...
  4. pokud máš víc druhu Javy, její verzi zjistíš v příkazovém řádku zadáním java -version
  5. přidej exe té javy v C:\Proram Files\java
